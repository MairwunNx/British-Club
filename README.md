## ðŸŽ¨ Creating Your Own Theme in Telegram

Want to customize or create a new theme from this one?

1. **Open Telegram Desktop** and apply this theme first
2. Go to **Settings** â†’ **Chat Settings**
3. Scroll down and click **Edit theme** (appears when custom theme is active)
4. This opens the Telegram Theme Editor at: https://t.me/addtheme/
5. Make your changes in the editor
6. Click **Create theme** when done
7. You'll get a unique theme link like: `t.me/addtheme/britishclub`
8. Share the link or export the `.tdesktop-palette` file

**Pro tip:** You can also edit the `.tdesktop-palette` file directly in any text editor to fine-tune colors!
